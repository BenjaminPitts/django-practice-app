## serve locally
### prerequisites: 
- [Python3](https://www.python.org/downloads/)
### in your terminal, run the following commands:
- `git clone https://github.com/BenjaminPitts/django-practice-app.git`
- `cd django-practice-app` 
- `python3 manage.py runserver`
